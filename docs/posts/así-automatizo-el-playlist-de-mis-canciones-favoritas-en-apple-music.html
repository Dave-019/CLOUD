Así automatizo el playlist de mis canciones favoritas en Apple Music

No sé si sabían, pero aquí en mi sitio tengo una página donde comparto las canciones a las que les he dado “fav” o “like” últimamente.

Básicamente, estoy escuchando música durante el día y uso mi Stream Deck para darle “fav” o “corazoncito” si una rola me gusta.

Las canciones son automáticamente listadas en esa página que mencioné arriba y si se meten, pueden ver la lista de mis rolas favoritas a la vez que pueden escuchar un preview de 30 segundos de cada una.

Todo bien hasta ahí, pero se me ocurrió que podría ofrecer un playlist de Apple Music que contuviera todas las canciones a las que le voy dando “fav”. ❤️

Nativamente y por default, Apple Music crea un playlist que va almacenando esas canciones, pero es un playlist que Apple Music maneja de manera especial y no permite que sea un playlist de acceso público. Peeero, descubrí que, al igual que el playlist nativo, los playlist inteligentes tampoco se pueden compartir públicamente. 😩 Entonces me puse a investigar más y vi que podría crear un playlist “espejo” de mi playlist inteligente, pero esta vez, como se trata de un playlist regular, podría configurarlo como público.

Entonces, para alimentar y copiar el contenido de mi playlist inteligente a mi playlist público regular, vi que podría usar Apple Shortcuts combinado con AppleScript.

Entonces la idea general es:

  1. Crear un Apple Shortcut
  2. Dentro, crear una acción AppleScript con el código que se encargue de todo
  3. Automatizar el shortcut para que corra todos los días a las 11pm y sincronice mis mi playlist inteligente con el playlist público

Y pues quiero compartir el proceso, para tenerlo documentado aquí.

Creando el Apple Shortcut

Dentro de Apple Shortcuts, lo que hice fue crear un nuevo shortcut, yo le di el nombre de “Sync Apple Music Favorites” y luego del lado derecho hay que buscar la acción “Run AppleScript” y lo agregamos a nuestro shortcut.

Luego, el código que hace toda la magia:

tell application "Music"
	set sourcePlaylistName to "Canciones Favoritas"
	set destinationPlaylistName to "Canciones Favoritas (Pública)"
	
	-- Verificar que existan las playlists
	try
		set sourcePlaylist to playlist sourcePlaylistName
	on error
		display notification "No se encontró la playlist origen: " & sourcePlaylistName with title "Apple Music Sync"
		return
	end try
	
	try
		set destinationPlaylist to playlist destinationPlaylistName
	on error
		display notification "No se encontró la playlist destino: " & destinationPlaylistName with title "Apple Music Sync"
		return
	end try
	
	-- Obtener identificadores únicos del destino (nombre + artista)
	set destinationKeys to {}
	repeat with t in (tracks of destinationPlaylist)
		try
			set trackKey to (name of t) & "||" & (artist of t)
			set end of destinationKeys to trackKey
		on error errMsg
			display notification "Error leyendo destino: " & errMsg with title "Apple Music Sync"
		end try
	end repeat
	
	-- Recorrer tracks de origen y agregar los que falten
	set addedCount to 0
	set errorCount to 0
	
	repeat with t in (tracks of sourcePlaylist)
		try
			set trackKey to (name of t) & "||" & (artist of t)
			
			if destinationKeys does not contain trackKey then
				try
					duplicate t to destinationPlaylist
				on error
					-- Fallback para tracks de iCloud/streaming
					add t to destinationPlaylist
				end try
				set end of destinationKeys to trackKey
				set addedCount to addedCount + 1
			end if
		on error errMsg
			set errorCount to errorCount + 1
		end try
	end repeat
	
	-- Notificación de resultado
	if errorCount > 0 then
		display notification "Errores en " & errorCount & " tracks. Agregadas: " & addedCount with title "Apple Music Sync ⚠️"
	else if addedCount is 0 then
		display notification "Tu playlist pública ya está al día ✅" with title "Apple Music Sync"
	else if addedCount is 1 then
		display notification "Se agregó 1 canción nueva 🎧" with title "Apple Music Sync"
	else
		display notification "Se agregaron " & addedCount & " canciones nuevas 🎧" with title "Apple Music Sync"
	end if
end tell

Lo que hace este código es, básicamente recorrer las canciones que existen en mi playlist inteligente y los copia a mi playlist público.

Hay dos parámetros importantes, que son los nombres de los playlist que tienen que coincidir exactamente, sino, este script no va a funcionar.

En mi caso se llaman así:

  • Playlist inteligente: Canciones Favoritas
  • Playlist público: Canciones Favoritas (Pública)

Al correr este script, se envía una notificación solamente si hay cambios en el playlist, y si durante todo el día no le di “fav” a ninguna canción, entonces no tiene caso mandar notificación.
